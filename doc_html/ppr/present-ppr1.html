NAME

present-ppr - run Portable Presenter perl-scripts.

SYNOPSIS

present-ppr [options] file

DESCRIPTION

Portable Presenter (PPresenter) is a package designed to give presentations. It is written in Perl/Tk only, which is available for UNIX and for Windows. Usually, you will run present which will call this program when the file supplied is a Perl program.

-help
-h
-?

Show brief help message.

SEE ALSO

A full documentation in html is included in the package, and available on the website: http://www.dhp.nl/~ppresenter/.
