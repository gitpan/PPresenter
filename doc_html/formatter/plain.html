Formatter
Plain 		Portable Presenter has a few formatters, which each enable someone to specify what text should appear on a slide. The plain formatter is a simplified version of the markup formatter: it displays text as it is entered.

The plain text format is simply putting a <PRE> container around the text as specified by the user, and then passes the text on to the markup formatter.

In most cases, the result shown will not satisfying because the font is fixed, which looks poor.

Plain formatter options

The plain formatter is a sub-class of the markup formatter, so has the same
  • general formatter options and
  • markup formatter options,
  • but no own options (yet).
