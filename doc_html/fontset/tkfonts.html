Portable Presenter has a few fontsets, which are used to show text on the slides. The fonts used by Tk's fontCreate function are non-scalable, which shows poor results on the fontsizes required in a presentation.

You can only use these fonts writing left-to-write. Help is required to teach the formatter to handle other directions.

Tk Fonts Options

-fixedFont => string
Name of the font to be used for the teletype characters. Default, this is Courier.

-proportionalFont => string
Name of the font to be used for the normal, proportional characters: normal running text. By default, this is Helvetica.
